# Podcast Listening Behavior Analysis

This project analyzes a dataset of podcast episodes using statistical summaries and visualizations to understand which factors may influence listening time (`Watch_Time_minutes`).

## Objectives
- Explore the dataset and understand its structure  
- Identify relationships between variables  
- Examine factors that may affect listening time  

## Data Processing
The analysis includes the following steps:
1. General statistical analysis of the dataset  
2. Identification of categorical and numerical variables  
3. Description of categorical variables  
4. Description of numerical variables  
5. Detection of missing values  
6. Handling missing values  
7. Detection and treatment of outliers  
8. Normalization of numerical variables  
9. Correlation analysis between variables  
10. Visualization of variable distributions  
11. Analysis of relationships with the target variable (`Watch_Time_minutes`)  

## Key Insights
- Episode length shows the strongest relationship with listening time  
- Most variables have weak influence on listening behavior
